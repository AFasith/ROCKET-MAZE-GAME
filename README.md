# Rocket Maze Game

Rocket Maze is an exciting maze game where you navigate a rocket through challenging mazes to reach the goal. Dodge obstacles, navigate narrow passages, and pilot your rocket to victory!

## Features
- Realistic rocket movement and controls.
- Beautifully designed mazes with dynamic obstacles.
- Engaging gameplay with challenging puzzles and obstacles.

## How to Play
1. Download or clone the repository.
2. Run `maze.py` using Python.
3. Use the arrow keys to control the rocket (Up, Down, Left, Right).
4. Navigate through the maze and reach the goal to win.

## Installation
Ensure you have Python installed on your system. No additional dependencies are required.
